# simppt

超簡單的 Markdown 簡報播放器 — **一個零依賴的 `index.html`，雙擊就能用**。

寫給「簡單文字、背景色、一點點圖」的簡報：不用裝任何東西、不用 build、不用連網。

## 使用

**開本機 server（推薦）**：

```sh
node server.js        # → http://localhost:4646（PORT 環境變數可換）
```

網址直接指定簡報：`http://localhost:4646/?deck=decks/templates.md` — 外部編輯器存檔**自動重載**、圖片相對路徑直接顯示，不用拖資料夾。

**或零安裝**：

1. 雙擊 `index.html` 打開瀏覽器
2. 按「**新簡報**」直接開始寫（左邊 markdown、右邊即時預覽），或按「看範例」、拖入你的 `.md`
3. 含圖片的簡報：把 `.md` 和圖片放同一個資料夾，**整個資料夾拖進來**
4. `F` 全螢幕開始播（或錄影）

## 內建編輯器

- 「**新簡報**」「**開啟檔案／資料夾**」都直接進編輯模式：左邊寫、右邊即時預覽**游標所在的那一頁**
- `Ctrl+S` 儲存：開啟的檔案直接**寫回原檔**；新簡報跳另存視窗（Chrome/Edge），其他瀏覽器自動下載 .md
- `Ctrl+Enter` 或「▶ 播放」從游標那頁開始播；播放中 `Esc` **回編輯器**，編輯器「首頁」鈕才離開
- **右邊預覽可以直接編排**（圖片和文字都行）：拖曳＝移動（**自動吸附**置中線與其他區塊、顯示紅色虛線）、右下角藍色把手＝縮放（圖片改寬度、文字改字級）、雙擊＝還原該塊；動完自動寫回 markdown（圖片 `![](){x= y= w=}`、文字行尾 `{x= y= s= c=}`，百分比座標），播放時同位置呈現
- **點一下文字塊**浮出迷你工具列：`B` 加粗、`A−`/`A＋` 字級、色票＋自訂色（寫回 `c=`）、等距、清除；Shift+點可多選批次操作。`s=`/`c=`/`b=` 也可以單獨手寫在行尾（例：`重點句 {s=130 c=#e94560}`）
- **句中個別字的樣式**：`{{文字|s=150 c=#e94560 b=1}}` — 只有這幾個字變大/變色/加粗，其餘照舊（例：`你唯一要 {{ALL IN|s=180 c=#e94560}} 的就是自己`）
- 「**還原版面**」鈕＝清掉**此頁**所有拖曳改動，回到 markdown 預設排版
- **直接在編輯區貼上圖片**（Ctrl+V 截圖）：自動上傳到 urusai.cc 圖床、插入 `![](https://i.urusai.cc/....png)`，任何模式都看得到圖（localhost 走 server 代理上傳，`/__upload`）
- 重整／關瀏覽器會**自動記住現場**（內容、模式、頁數、開啟的檔案），回來繼續編；「首頁」鈕才會清掉

用「開啟檔案／開啟資料夾」按鈕載入後直接播放的話（Chrome/Edge），外部編輯器存檔畫面也會**自動重新載入**。

## 簡報語法

```md
---
bg: #1a1a2e        ← 檔首 frontmatter：全域背景色 / 文字色
color: #ffffff
---

# 第一頁標題
簡單文字內容

--- bg=#0f3460     ← 分隔線同行 key=value，覆蓋這一頁的設定

## 第二頁
- 支援列表
- **粗體**、*斜體*、`程式碼`

---

![](img/pic.png)   ← 圖片自動置中縮放
```

- `---` 獨立一行＝分頁（恰好三個 dash）
- 支援：`#`/`##`/`###` 標題、無序/有序列表、圖片、粗斜體、行內 code、連結、段落
- `accent: #d84a3f`（frontmatter 或分隔線 `accent=`）：**粗體**自動帶這個顏色，標重點用
- `--- layout=top`：標題在上、內文在下、左對齊的版型（預設是置中）；現成版型見 `decks/templates.md`
- `--- layout=split`：**左圖右文** — 該頁的圖片固定在左半邊，標題和文字排右邊（圖寫在哪一行都行，會自動抓去左邊）
- `--- set bg=#ffffff color=#1c1c1e`：**從這頁起**都用這組設定（例如前幾張黑底白字、後面全部白底黑字，切一次就好）；不加 `set` 只作用那一頁，下一頁自動回到原本設定
- `width: 95`（frontmatter 全域）或 `--- width=95`（單頁）：內容行寬，佔螢幕寬的百分比（預設 84），字太快換行就調大

## 操作

| 按鍵 | 功能 |
|---|---|
| `←` `→` `Space` | 翻頁（點擊畫面也可以，左側 1/4 = 上一頁） |
| `Home` / `End` | 第一頁 / 最後一頁 |
| `F` | 全螢幕 |
| `Esc` / 右上 ✕ | 結束播放回編輯器（全螢幕時先退全螢幕，再按一次退播放） |
| `P` | 顯示/隱藏頁碼（預設隱藏，錄影畫面乾淨） |
| `N` | 顯示/隱藏**演講稿**（md 裡 `note: 想講的話` 開頭的行，觀眾/錄影看不到內容本體） |
| `D` | **畫筆模式**（粗筆直接畫在投影片上；再按 D 或 Esc 離開） |
| `X` | 清除畫筆筆跡（換頁也會自動清） |
