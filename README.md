# simppt

超簡單的 Markdown 簡報播放器 — **一個零依賴的 `index.html`，雙擊就能用**。

寫給「簡單文字、背景色、一點點圖」的簡報：不用裝任何東西、不用 build、不用連網。

## 使用

1. 雙擊 `index.html` 打開瀏覽器
2. 按「**新簡報**」直接開始寫（左邊 markdown、右邊即時預覽），或按「看範例」、拖入你的 `.md`
3. 含圖片的簡報：把 `.md` 和圖片放同一個資料夾，**整個資料夾拖進來**
4. `F` 全螢幕開始播（或錄影）

## 內建編輯器

- 「新簡報」或播放中按 `E` 進入編輯模式：左邊寫、右邊即時預覽**游標所在的那一頁**
- `Ctrl+S` 儲存：用「開啟」按鈕載入的檔案直接**寫回原檔**；新簡報跳另存視窗（Chrome/Edge），其他瀏覽器自動下載 .md
- `Ctrl+Enter` 或「▶ 播放」從游標那頁開始播

用「開啟檔案／開啟資料夾」按鈕載入後直接播放的話（Chrome/Edge），外部編輯器存檔畫面也會**自動重新載入**。

## 簡報語法

```md
---
bg: #1a1a2e        ← 檔首 frontmatter：全域背景色 / 文字色
color: #ffffff
---

# 第一頁標題
簡單文字內容

--- bg=#0f3460     ← 分隔線同行 key=value，覆蓋這一頁的設定

## 第二頁
- 支援列表
- **粗體**、*斜體*、`程式碼`

---

![](img/pic.png)   ← 圖片自動置中縮放
```

- `---` 獨立一行＝分頁（恰好三個 dash）
- 支援：`#`/`##`/`###` 標題、無序/有序列表、圖片、粗斜體、行內 code、連結、段落
- `accent: #d84a3f`（frontmatter 或分隔線 `accent=`）：**粗體**自動帶這個顏色，標重點用
- `--- layout=top`：標題在上、內文在下、左對齊的版型（預設是置中）；現成版型見 `decks/templates.md`

## 操作

| 按鍵 | 功能 |
|---|---|
| `←` `→` `Space` | 翻頁（點擊畫面也可以，左側 1/4 = 上一頁） |
| `Home` / `End` | 第一頁 / 最後一頁 |
| `F` | 全螢幕 |
| `P` | 顯示/隱藏頁碼（預設隱藏，錄影畫面乾淨） |
