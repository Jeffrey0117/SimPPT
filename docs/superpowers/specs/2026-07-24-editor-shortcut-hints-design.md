# 編輯器快捷鍵提示

2026-07-24 定案。編輯器陸續加了 Ctrl+H（搜尋/取代）、Ctrl+方向鍵（換頁）這些快捷鍵，但沒有任何地方顯示出來，使用者容易忘記這些功能存在、誤以為沒做。

## 決策脈絡

- 兩處都要更新：頂部工具列的 `#ed-status`（已經過時，只列 Ctrl+S/Ctrl+Enter）+ 右側預覽區底下新增一行提示（緊接在 `#ed-pageinfo` 頁碼文字下面，使用者盯著簡報預覽時視線範圍內）。
- 兩處內容一致，都列現有四組：`Ctrl+S 儲存`、`Ctrl+Enter 播放`、`Ctrl+H 搜尋取代`、`Ctrl+方向鍵 換頁`。

## 變更

1. `#ed-status`（`#editor-bar` 內既有 `<span>`）文字更新，加上 Ctrl+H、Ctrl+方向鍵。
2. `#ed-preview` 內、`#ed-pageinfo` 之後新增 `<div id="ed-shortcuts">`，樣式比照原本 `#ed-pageinfo` 的灰階小字（`12.5px; color: #77777f`，跟頁碼文字做出主次區分——頁碼是白色大字的主資訊，快捷鍵是次要提示）。

## 測試範圍

純文字/CSS，不寫自動測試；語法檢查 + 手動瀏覽器確認兩處文字都正確顯示。

## 不做（YAGNI）

不做互動式的「?」說明彈窗、不做快捷鍵自訂、不列播放模式（非編輯器）的快捷鍵（那些是另一個情境，不在這次範圍）。
